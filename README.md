# responsive-dev-portfolio-with-css
LefaDev's personal portfolio – a showcase of modern HTML/CSS techniques including flexbox layouts, CSS animations, and mobile-first design. Clean, fast, and recruiter-ready.
# Features

- **Sticky Glass Navbar** – Blur effect with smooth scroll links
- **Animated Hero Section** – Floating profile image with fade-in text
- **Scroll Reveal Animations** – Content fades in as you scroll
- **Interactive Cards** – Hover effects with social links
- **Mobile Hamburger Menu** – Fully responsive navigation
- **Social Media Integration** – GitHub, LinkedIn, email, Twitter
- **Custom Scrollbar** – Branded scrollbar styling
- **Stats Section** – Visual credibility indicators
- **CTA Banner** – Eye-catching contact section

# Built With

- HTML5
- CSS3 (Flexbox, Grid, Animations, Glass-morphism)
- Vanilla JavaScript (Intersection Observer, Mobile Menu)
- Font Awesome Icons
